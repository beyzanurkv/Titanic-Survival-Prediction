# Titanic Survival Prediction

This project uses the Titanic dataset to predict whether a passenger survived or not.

## What I did
- Loaded the train and test data.
- Combined the datasets for preprocessing.
- Removed some columns that were not useful.
- Extracted titles from names.
- Filled missing values in columns like Age and Fare.
- Converted categorical columns into numeric form.
- Scaled the data.
- Split the data into train and test sets.
- Tested different machine learning models.

## Best model
The best model was Logistic Regression.

## Result
The model reached about 0.82 accuracy.

## Used libraries
- pandas
- scikit-learn
